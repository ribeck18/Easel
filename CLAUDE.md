## Agent skills

### Issue tracker

Issues and PRDs are tracked as GitHub issues in `ribeck18/Easel` via the `gh` CLI. See `docs/agents/issue-tracker.md`.

Uses the five canonical triage labels as-is (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context: one `CONTEXT.md` + `docs/adr/` at the repo root. See `docs/agents/domain.md`.
